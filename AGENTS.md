# About this project

Basic app-shell layout (sidebar + header + main + breadcrumbs) lifted from
`wildcat-webapp`, stripped of its backend wiring: no auth, permissions,
billing/credits, Supabase, AI drawer, voice, or price ticker. The chrome runs
standalone with no environment or services required.
