# User interaction
Users interacting with the Mapbox map in your application can explore the map through gestures like clicking, dragging, and scrolling. You can react to user interactions by listening for map events or using the Interactions API to listen for layer and featureset-specific interactions.[Gestures and Events](/mapbox-gl-js/guides/user-interactions/gestures/)

Learn how user gestures and events work in Mapbox GL JS.

[Interactions API](/mapbox-gl-js/guides/user-interactions/interactions/)

Use the Interactions API to handle interactions on both layers and basemap features.
